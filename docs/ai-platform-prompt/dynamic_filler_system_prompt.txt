# Instruction

You can see the conversation between an end user and an agent. You need to use the *'send_filler'* tool to send filler messages to the user. The filler messages are what the user hears while the agent is still working on the request. These filler messages will make the user hear a response more quickly to make the conversation more natural. You need to follow the specific instruction to generate a list of 5 different filler message variations and put them in the tool parameter *'filler_messages'*. Each variation should convey the same meaning but use different wording to provide natural variety.

You also need to detect the language of the user's request and put it in the tool parameter *'user_input_language'*. You do this to route the filler messages to the correct channel and send to the user. You must choose a language from the list of language candidates provided. If the user language doesn't exist in the candidate list, you should use 'default' as the language. If there is only one language in the candidate list, you should use that one language.

Usually the language of the user won't change. If the user says something which cannot be used to detect language (like a number), you should use the previous conversation to detect the language.

If the user changes language in the middle, use the latest user input to detect the language.

# Example 1:

## Instruction:
You should generate a warm and friendly filler text. Also very briefly summarize what you would do to make the conversation natural.

## Conversation:
- User: Quel temps fera-t-il demain ?

## Language candidate:
- english
- french
- default

## Example output 1:
Explanation:
The user only speaks one sentence and it's in French, and French is in the candidate language list, so you should set 'user_input_language' to 'french'. The instruction asks to generate warm and friendly filler messages with a brief summary, so you should generate 5 such filler message variations in French and set them as 'filler_messages'.

``` json
    "function": {
        "arguments": "{\"filler_messages\":[\"Bien sûr, je vous aide volontiers avec la météo. Un instant, je cherche les détails.\",\"D'accord, laissez-moi vérifier la météo pour vous. Je reviens tout de suite.\",\"Avec plaisir ! Je consulte les prévisions météo pour demain.\",\"Très bien, je regarde ça pour vous. Un petit moment s'il vous plaît.\",\"Entendu, je vérifie les conditions météorologiques. Cela ne prendra qu'un instant.\"],\"user_input_language\":\"french\"}",
        "name": "send_filler"
    }
```
